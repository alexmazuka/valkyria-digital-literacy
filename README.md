# ⚔️ Українська Валькірія — Школа цифрової грамотності для жінок-ВПО

Інтерактивна онлайн-лекція (≈60 хвилин) для жінок — внутрішньо переміщених осіб. Навчає користуватися державними цифровими сервісами України самостійно, без черг і посередників.

## 📖 Зміст лекції

| Час | Модуль |
|----|--------|
| 5′ | **Вступ** — що робить «Українська Валькірія» для жінок-переселенок |
| 15′ | **Модуль 1** — «Дія»: застосунок, портал, цифрові документи |
| 10′ | **Модуль 1.2** — ЕЦП / КЕП: електронний підпис, Дія.Підпис |
| 10′ | **Модуль 2.1** — електронні послуги для ВПО, довідка ВПО |
| 15′ | **Модуль 2.2** — соцвиплати, житлові субсидії, допомога ВПО онлайн |
| 5′ | **Практикум** — закріплення, чек-листи, підсумок |

Плюс довідкові розділи: корисні ресурси й література, законодавчі та нормативні акти, громадські ініціативи та рухи для жінок.

## 🎯 Інтерактив

- ⏱ Таймер лекції на 60 хвилин
- ✅ 3 квізи з миттєвою перевіркою
- 🎮 Симулятор електронного підпису (Дія.Підпис)
- 🎒 Персональні чек-листи (зберігаються у браузері)
- 📊 Трекер прогресу
- ✍️ Вправа-рефлексія

## 🚀 Запуск

Сайт статичний. Відкрийте `index.html` у браузері або задеплойте через **GitHub Pages** (Settings → Pages → Deploy from branch → `main` / root).
